Projects

Study Buddy

The SuddyBuddy site is a JavaEE site running on a Tomcat 8 server. It is a tool that allows the user to create customized repositories of study questions and tips. Study material can be reviewed via reviews sessions which can also be customized based on various criteria. Technologies used include JavaEE 7, Apache Tomcat 8, JSP, JSTL, MySQL, HTML, CSS, and the Bootstrap framework.
